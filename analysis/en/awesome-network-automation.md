# Awesome Network Automation - Analysis

> **Repository**: [networktocode/awesome-network-automation](https://github.com/networktocode/awesome-network-automation)

## Overview

Resource collection at the intersection of network infrastructure and programming. Covers IaC methodologies, vendor APIs, data modeling standards, and enterprise-scale automation workflows.

## Main Categories

| Category                 | Description                                            |
| ------------------------ | ------------------------------------------------------ |
| Programming Topics       | YAML, JSON, YANG, Jinja2, RESTCONF/NETCONF, Python, Go |
| Configuration Management | Ansible, Salt, Puppet, Chef                            |
| Network-Specific Tools   | Batfish, NAPALM, Netmiko, Nornir                       |
| Network Telemetry        | Telemetry collection and monitoring                    |
| Emulators & Simulators   | Network simulation environments                        |

## Key Projects

- **Ansible** — Configuration management framework
- **Batfish** — Network configuration analysis
- **NAPALM** — Multi-vendor abstraction
- **Nornir** — Python-based task execution
- **StackStorm** — Event-driven automation

## Relevance & Advantages for autonomously-io

Network automation is a key building block for autonomous operations in telecom environments. Vector collects network telemetry, and combining these automation tools with AI-driven decision-making from the autonomously-io platform enables closed-loop network management.
