# Awesome Telco - Analysis

> **Repository**: [ravens/awesome-telco](https://github.com/ravens/awesome-telco)

## Overview

Open source project catalog covering the complete mobile network stack — from SIM cards to 5G core, RAN, security, and lab environments.

## Main Categories

| Category             | Description                                  |
| -------------------- | -------------------------------------------- |
| SIM Cards            | PySIM, sysmoISIM, eSIM/eUICC management      |
| User Equipment       | 4G/5G modems, diagnostics (QCSuper, SCAT)    |
| Radio Access Network | O-RAN, 5G gNodeB, 4G eNodeB, PHY layer tools |
| Core Network         | 5GC (Open5GS, free5GC, OAI), EPC, UPF        |
| Interconnect         | IMS, SBC, SS7, Diameter                      |
| Infrastructure       | NFV, SDN, Kubernetes deployment              |
| Orchestration        | Automation and deployment tools              |
| Lab & Testbeds       | Ready-to-use test environments               |
| Security             | Research, detection, vulnerability analysis  |

## Key Projects

- **Open5GS** — Open source 5G core
- **free5GC** — 5GC implementation (Go)
- **UERANSIM** — 5G UE/RAN simulator
- **srsRAN** — Software-defined radio RAN

## Relevance & Advantages for autonomously-io

Telecom networks generate massive telemetry from RAN, Core, and Transport layers. Vector collects this data, OpenObserve analyzes and visualizes it, and Keep orchestrates automated responses — forming the foundation for telecom autonomous operations.
