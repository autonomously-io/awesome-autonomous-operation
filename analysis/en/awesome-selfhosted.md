# Awesome Self-Hosted - Analysis

## Overview

Catalog of 80+ categories of self-hostable open-source software, focusing on privacy-respecting alternatives to commercial SaaS.

## Categories Relevant to Autonomous Operations

| Category             | Key Tools                                    |
| -------------------- | -------------------------------------------- |
| Analytics            | Metabase, Redash, PostHog                    |
| Automation           | Huginn, Apache Airflow, StackStorm, Cronicle |
| Monitoring           | Healthchecks (ping monitoring + alerting)    |
| Software Development | CI/CD, version control, project management   |

## Relevance & Advantages for autonomously-io

Autonomous operation platforms are often deployed on-premises or in private cloud environments. OpenObserve itself is a self-hostable observability platform that fits this context. The automation tools listed here (StackStorm, Airflow) complement Keep's workflow engine for broader operational automation.
