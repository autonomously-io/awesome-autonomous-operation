# Awesome Telco Cloud 分析

> **Repository**: [LesFacilitateurs/awesome-telco-cloud](https://github.com/LesFacilitateurs/awesome-telco-cloud)

## 概要

テレコムのクラウドネイティブ変革に焦点を当てたリソース集。コンテナ化 NF、Kubernetes オーケストレーション、標準リファレンスアーキテクチャを網羅。

## 主要カテゴリ

| カテゴリ                      | 内容                                              |
| ----------------------------- | ------------------------------------------------- |
| Resources                     | 書籍、コース、ハンズオンラボ、論文、ブログ        |
| NF Development                | Ligato, TRex, Firecracker 等の NF 開発ツール      |
| NF Orchestration & Automation | マルチファンデーション対応                        |
| Open Source Cellular NFs      | 2G〜5G 実装                                       |
| Private Networks              | 商用ソリューション（BubbleRAN, OAIBOX, firecell） |
| AI for Telcos                 | 3GPP ドキュメント向け Telco-RAG                   |

## autonomously-io との関連性・優位性

クラウドネイティブなテレコムインフラは autonomously-io のデプロイ対象。Kubernetes 上の CNF は Vector + OpenObserve に適した構造化テレメトリを生成。Telco-RAG は autonomously-io が Qdrant で目指す運用知識検索と同じパターン。
