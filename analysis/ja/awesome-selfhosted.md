# Awesome Self-Hosted 分析

> **Repository**: [awesome-selfhosted/awesome-selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted)

## 概要

80以上のカテゴリにわたるセルフホスト可能な OSS のカタログ。商用 SaaS の代替としてプライバシー重視のソリューションを集約。

## 自律運用に関連するカテゴリ

| カテゴリ             | 主要ツール                                   |
| -------------------- | -------------------------------------------- |
| Analytics            | Metabase, Redash, PostHog                    |
| Automation           | Huginn, Apache Airflow, StackStorm, Cronicle |
| Monitoring           | Healthchecks（Ping 監視 + アラート）         |
| Software Development | CI/CD, バージョン管理, プロジェクト管理      |

## autonomously-io との関連性・優位性

自律運用基盤はオンプレミス/セルフホスト環境での運用が前提となるケースが多い。OpenObserve 自体がセルフホスト可能な Observability プラットフォームとしてこの文脈に位置する。StackStorm や Airflow は Keep のワークフローエンジンを補完。
