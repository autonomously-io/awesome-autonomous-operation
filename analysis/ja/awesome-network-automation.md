# Awesome Network Automation 分析

> **Repository**: [networktocode/awesome-network-automation](https://github.com/networktocode/awesome-network-automation)

## 概要

ネットワークインフラとプログラミングの交差領域に特化したリソース集。IaC、ベンダー API、データモデリング標準、エンタープライズ規模の自動化ワークフローを網羅。

## 主要カテゴリ

| カテゴリ                 | 内容                                                   |
| ------------------------ | ------------------------------------------------------ |
| Programming Topics       | YAML, JSON, YANG, Jinja2, RESTCONF/NETCONF, Python, Go |
| Configuration Management | Ansible, Salt, Puppet, Chef                            |
| Network-Specific Tools   | Batfish, NAPALM, Netmiko, Nornir                       |
| Network Telemetry        | テレメトリ収集と監視                                   |
| Emulators & Simulators   | ネットワークシミュレーション環境                       |

## 主要プロジェクト

- **Ansible** — 構成管理フレームワーク
- **Batfish** — ネットワーク構成分析
- **NAPALM** — マルチベンダー抽象化
- **Nornir** — Python ベースタスク実行
- **StackStorm** — イベント駆動自動化

## autonomously-io との関連性・優位性

テレコム環境における自律運用の重要な構成要素。Vector によるネットワークテレメトリ収集と、これらの自動化ツールの AI 駆動による意思決定の組み合わせが、クローズドループなネットワーク管理を実現。
