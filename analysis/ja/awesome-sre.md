# Awesome SRE 分析

> **Repository**: [dastergon/awesome-sre](https://github.com/dastergon/awesome-sre)

## 概要

SRE の文化・プラクティス・ツールの包括的コレクション。Google の SRE 書籍を起点に、Netflix, LinkedIn, Uber 等の実践を網羅。

## 主要カテゴリ

| カテゴリ                   | 内容                               |
| -------------------------- | ---------------------------------- |
| Culture                    | SRE の哲学と組織的アプローチ       |
| Reliability                | 耐障害性、テスト、システム設計     |
| Monitoring & Observability | 監視・検知プラクティス             |
| On-Call                    | インシデント対応と運用の持続可能性 |
| Post-Mortem                | 障害からの学習                     |
| Capacity Planning          | リソース予測                       |
| Service Level Agreement    | SLO/SLI/SLA 定義                   |
| SRE Tools                  | 実用ツール                         |

## autonomously-io との関連性・優位性

自律運用は SRE の進化形。SRE エンジニアが手動で行ってきたインシデント対応・キャパシティプランニング・ポストモーテム・トイル削減を AI で自動化する。Keep のワークフロー自動化が SRE のトイルに、OpenObserve が SLI/SLO 計測基盤として対応。
