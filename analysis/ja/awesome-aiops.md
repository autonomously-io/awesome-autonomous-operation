# Awesome AIOps 分析

> **Repository**: [linjinjin123/awesome-AIOps](https://github.com/linjinjin123/awesome-AIOps)

## 概要

AIOps（Artificial Intelligence for IT Operations）に特化したリソース集。中国テック企業（Tencent, Alibaba, Baidu）の実践事例と学術研究が豊富。

## 主要カテゴリ

| カテゴリ              | 内容                                                               |
| --------------------- | ------------------------------------------------------------------ |
| White Papers          | エンタープライズ向け AIOps 導入ガイダンス                          |
| Educational Content   | 清華大学コース、ML 運用教材                                        |
| Industry Case Studies | Tencent, Alibaba, Baidu, Yahoo, Netflix, LinkedIn, Uber の実装事例 |
| Tools & Algorithms    | 時系列分析、異常検知フレームワーク                                 |
| Research Papers       | 根本原因分析、KPI クラスタリング、局所化                           |
| Public Datasets       | ベンチマークデータセット                                           |

## 主要プロジェクト

- **Tencent Metis** — インテリジェント運用プラットフォーム
- **Baidu Curve** — 時系列ラベリングツール
- **Microsoft TagAnomaly** — 時系列アノテーション
- **Facebook Prophet** — 予測ライブラリ
- **Google CausalImpact** — 時系列因果推論
- **Yahoo EGADS** — 異常検知
- **tsfresh** — 特徴量抽出ライブラリ

## autonomously-io との関連性・優位性

AIOps は自律運用の中核概念。異常検知・根本原因分析・アラート削減は OpenObserve + Vector パイプラインで収集したテレメトリに対する分析レイヤーに直結。Keep の AI 相関機能が AIOps アルゴリズムと運用ワークフローを橋渡しする。
